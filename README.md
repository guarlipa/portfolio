
# Next-JS Portfolio

Repository with the resources developed for the creation of my professional portfolio.


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
## Deployment

To deploy this project run


```bash
  npm run dev
```
